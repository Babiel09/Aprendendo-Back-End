<div>
  <h2>Primeiramente baixe o zip da aplicação:</h2>
  <ul>
    <li>Aperte o botão "Code"</li>
    <li>Vá até "Download Zip"</li>
    <li>Baixe o arquivo .zip</li>
  </ul>
